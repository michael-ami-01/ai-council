## How It Works

1. **Enter any question** — career decisions, technical debates, ethical dilemmas, business strategy
2. **AI Council deliberates** — 18 experts analyze in batches of 3, building on each other's perspectives
3. **Synthesized verdict** — aggregated consensus with confidence levels
